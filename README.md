# Human Atlas

Human Atlas is a data-driven web application for exploring human evolution, migration patterns, and genetic relationships. It combines a relational database with geospatial and graph-based data models to visualize species history, movement routes, and gene flow between populations.

## Overview

The project is built around three core data domains:

* Species evolution timeline and taxonomy
* Human migration routes with geospatial waypoints
* Genetic admixture and gene flow networks


## Tech Stack

* Next.js (App Router, TypeScript)
* Neon PostgreSQL (serverless database)
* Drizzle ORM (SQL integration layer)
* PostGIS (geospatial queries)
* React Query (client data fetching)
* Zustand (state management)
* D3.js / Deck.gl (data visualization)
